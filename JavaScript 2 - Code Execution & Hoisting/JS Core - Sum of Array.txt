				QUESTION

Write a function that calculates the sum of all numbers in an array.

Example:
	Input:	[1, 2, 3, 4]

	Output:	10
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
				SOLUTION

function sumOfArray(arr) {
  	let sum = 0;
  	for(let i=0;i<arr.length;i++) {
      		sum = sum + arr[i];
  	}
  	return sum;
}
